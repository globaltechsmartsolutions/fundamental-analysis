# Tests y Scripts de Prueba

Scripts de prueba y comparación para el sistema.

## Contenido

- **test_config.py** - Prueba configuración del sistema
- **test_target_fetcher.py** - Prueba obtención de valores objetivo
- **scrape_and_save.py** - Script de scraping y guardado de valores

## Uso

Estos scripts son principalmente para desarrollo y pruebas. No son parte del flujo principal del sistema.
